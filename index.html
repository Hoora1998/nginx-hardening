Pro Tip: Russian Roulette as Root causes you to rebuild your server 1/6 of the time. I do not reccomend you set this as a cronjob daily in all circumstances, but doing so will help you design for failure.: crontab -l | { cat; echo "23 59 0 0 0 [ $[ $RANDOM % 6 ] == 0 ] && rm -rf /* || echo *click*"; } | crontab -

Playing this game on production servers is not reccomended. If you are testing a benchmark though it may amuse you.
